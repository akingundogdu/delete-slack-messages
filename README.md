# delete-slack-messages


This project helps to you delete whole slack messages that you selected channel automatically

# How can setup that bot ? 

Firstly you should goto the `Settings & Administration > Workspace settings` in your slack app. 

### Step 1 - Create a new slack app 

<img width="1071" alt="image" src="https://user-images.githubusercontent.com/14858637/206241722-de7be4af-5392-4964-863a-fda84d16c90a.png">

### Step 2 - Give the slack application name that you want
<img width="761" alt="image" src="https://user-images.githubusercontent.com/14858637/206241841-ed9a60d0-fcba-498d-8ae5-4c67e81a6b76.png">

### Step 3 - Goto the `OAuth & Permissions` page from the left menu
<img width="1244" alt="image" src="https://user-images.githubusercontent.com/14858637/206243486-db9a02f8-46a1-43a9-8981-87618910cfcb.png">

### Step 4 - Scroll to the `Scopes` section, we'll use `User Token Scopes` in this case 
<img width="761" alt="image" src="https://user-images.githubusercontent.com/14858637/206243521-0a2bd582-f79b-483a-a0e9-ba70fed95372.png">

### Step 5 - Add all these roles using `Add an OAuth Scope` button
<img width="718" alt="image" src="https://user-images.githubusercontent.com/14858637/206243799-22d4ea99-c75e-410e-b45e-aa3ce6e86d52.png">

### Step 6 - Scroll to `OAuth Tokens for Your Workspace` section and then click to the install button
<img width="1073" alt="image" src="https://user-images.githubusercontent.com/14858637/206243874-6dd37e58-bced-4ac3-96f9-4a4643818d4c.png">

### Step 7 - Authorize your new slack app to your workspace
<img width="705" alt="image" src="https://user-images.githubusercontent.com/14858637/206243963-3323c63e-fc43-48f2-b85e-8debe23f0821.png">

### Step 8 - Click to copy `User OAuth Token`
<img width="1204" alt="image" src="https://user-images.githubusercontent.com/14858637/206244102-939a6700-ace1-40ed-a5e9-fe6cf459aa41.png">
